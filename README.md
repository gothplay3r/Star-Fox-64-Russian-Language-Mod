# Russian Language Mod

Русская локализация для Star Fox 64: Recompiled.

* Автор: gothplay3r
* Версия: 1.0.0
* Сайт автора: https://boosty.to/gothplay3r
* Репозиторий: https://github.com/gothplay3r/Star-Fox-64-Russian-Language-Mod

Мод переводит игровой текст и основные графические элементы интерфейса на русский язык.

## Установка

Скачайте `Russian_Language_Mod_v1_0_0.nrm` из релиза и поместите его в папку модов Star Fox 64: Recompiled.
