# Russian Language Mod

Русская локализация для Star Fox 64: Recompiled.

* Автор: gothplay3r
* Версия: 1.0.0
* Сайт автора: https://boosty.to/gothplay3r
* Исходный код: https://github.com/gothplay3r/Star-Fox-64-Russian-Language-Mod

Мод переводит игровой текст и основные графические элементы интерфейса на русский язык.

## Установка

Установите мод через Thunderstore Mod Manager или скачайте `.nrm` из пакета вручную.
